# Modelo de missao

Copie este modelo para iniciar uma missao SuperDev 2.0.

## 00-🧭-PROCESSO

Use quando nao souber qual etapa chamar ou quando precisar retomar uma missao.

```text
Use:
00-🧠-SUPERDEV
+
00-🧭-PROCESSO

PEDIDO:
[explique o que voce quer fazer ou onde parou]
```

## 01-💡-IDEIA

```text
Use:
00-🧠-SUPERDEV
+
01-💡-IDEIA

GOAL:
[objetivo de negocio]
```

## 02-📋-PLANO

```text
Use:
00-🧠-SUPERDEV
+
02-📋-PLANO

Especificacao aprovada:
[cole a especificacao aprovada]
```

## 03-🔨-CONSTRUIR

```text
Use:
00-🧠-SUPERDEV
+
03-🔨-CONSTRUIR

Plano aprovado:
[cole o plano aprovado]
```

## 04-🧪-TESTAR

```text
Use:
00-🧠-SUPERDEV
+
04-🧪-TESTAR

Entrega para validar:
[cole resumo, diff ou arquivos alterados]
```

## 05-📦-ENTREGAR

```text
Use:
00-🧠-SUPERDEV
+
05-📦-ENTREGAR

Missao:
[cole objetivo, plano, implementacao e validacao]
```
