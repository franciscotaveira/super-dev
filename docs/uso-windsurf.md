# Uso no Windsurf

## Como chamar

No Cascade/Windsurf, referencie os arquivos da etapa:

```text
Use:
skills/00-🧠-SUPERDEV.md
skills/01-💡-IDEIA.md

GOAL:
[seu objetivo de negocio]
```

## Fluxo recomendado

- Comece por `01-💡-IDEIA`.
- Aprove a especificacao.
- Passe para `02-📋-PLANO`.
- Aprove o plano.
- So entao use `03-🔨-CONSTRUIR`.
- Depois valide com `04-🧪-TESTAR`.
- Feche com `05-📦-ENTREGAR`.

## Cuidados

- Se o agente editar arquivos durante IDEIA ou PLANO, interrompa e volte ao checkpoint.
- Se o agente propor refatoracao fora do objetivo, peca para marcar como "fora do plano".
