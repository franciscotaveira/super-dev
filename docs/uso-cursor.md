# Uso no Cursor

## Como chamar

No chat do Cursor, referencie os arquivos:

```text
Use:
@skills/00-🧠-SUPERDEV.md
@skills/01-💡-IDEIA.md

GOAL:
[seu objetivo de negocio]
```

## Para construir

Depois de aprovar o plano:

```text
Use:
@skills/00-🧠-SUPERDEV.md
@skills/03-🔨-CONSTRUIR.md

Plano aprovado:
[cole o plano]
```

## Dica operacional

Use contexto selecionado com cuidado. Inclua apenas arquivos relacionados a missao para evitar alteracoes fora do escopo.
